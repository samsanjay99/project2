# Project 2: Recipe Page

## Learning Objectives

### HTML Concepts:
1. **Lists**: Use `<ul>` for ingredients and `<ol>` for instructions
2. **Headings**: Properly structure content with `<h1>`, `<h2>`, `<h3>`
3. **Divs and Spans**: Use `<div>` for sections and `<span>` for inline styling

### CSS Concepts:
1. **CSS Selectors**: Use element, class, and ID selectors
2. **Positioning**: Apply relative and absolute positioning
3. **Box Shadows**: Add depth with box-shadow property
4. **Media Queries**: Make the page responsive for mobile devices

## Instructions

1. Create recipe structure in `index.html`:
   - Add a main heading (h1) for recipe name
   - Create an ingredients section with unordered list
   - Create an instructions section with ordered list
   - Use divs to organize sections
   - Add spans for special emphasis

2. Style the page in `styles.css`:
   - Use different selectors to style elements
   - Add box shadows to the container
   - Position elements (e.g., badge or image)
   - Add media queries for mobile responsiveness

## Expected Result
A well-structured recipe page with:
- Clear heading hierarchy
- Styled ingredient and instruction lists
- Nice shadows and spacing
- Responsive design for mobile
